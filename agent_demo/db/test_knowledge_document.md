# 贷款产品客服知识库测试文档

## 公司与产品介绍

我们是本土化运营的正规服务团队，主要为有真实资金需求、信用记录良好的用户提供短期资金周转服务。平台重视长期信用关系，用户按时还款、资料真实完整时，后续有机会获得更高额度和更低利率。

平台不会骚扰用户的家属和朋友，也不会采用威胁、恐吓、辱骂等不合规催收方式。如果用户遇到疑似违规沟通，可以保留截图并联系人工客服核实。

## 额度为什么不高

初始额度较低，通常是因为平台需要控制欺诈风险。系统会根据用户资料完整度、历史信用记录、设备环境、还款能力和实时风控策略综合评估。

如果用户是首次申请，系统一般会先给出较保守的额度。只要用户保持资料真实、按时还款、避免频繁更换设备或异常登录，后续额度可能逐步提升。

## 复借被拒原因

还款后再次申请被拒，并不一定代表用户信用差。复借审核会重新评估用户当前资料、历史还款表现、风险策略、授信额度池和产品政策。

如果用户刚完成还款，可以建议稍后再尝试。若多次复借失败，可以引导用户检查资料是否完整、手机号是否正常、银行卡状态是否有效，并等待系统后续重新评估。

## 利率如何降低

产品利率会根据用户风险等级、信用表现和平台策略综合确定。初期利率可能相对保守，后续会根据还款表现动态调整。

对于按时还款、信用表现稳定的用户，通常在多次良好还款记录之后，平台可能降低利率，并同步提高可用额度。具体结果以系统审核为准，人工客服无法手动承诺固定降息。

## 还款与逾期说明

用户应在账单到期日前完成还款。还款成功后，系统通常会自动更新账单状态。如果用户已经付款但页面仍显示未还款，可以建议用户提供支付时间、支付渠道和订单截图，由人工客服进一步核实。

如果发生逾期，用户可能会产生额外费用，也可能影响后续额度和复借结果。建议用户尽快处理账单，并保持沟通畅通。

## 隐私与资料安全

客服不应要求用户在公开聊天中发送完整身份证号、完整银行卡号、验证码或支付密码。如果用户主动发送隐私信息，系统应尽量脱敏后再进入后续处理。

用户可以提供必要的订单号、注册手机号后四位或还款截图，以便人工客服核实问题。验证码、密码和完整银行卡信息不应通过聊天窗口发送。

## 需要人工处理的情况

以下情况建议转人工客服处理：用户明确表示投诉、报警、起诉、被骗、高利贷、暴力催收、骚扰家人、泄露隐私、还款不到账或情绪明显激动。

转人工时，需要给人工客服留私有备注，说明用户诉求、命中的风险关键词、用户原文和建议下一步处理方式。

## 标准回复口径

当用户只发送“你好”“在吗”“有人吗”等低价值消息时，可以先礼貌询问用户想咨询的问题，并发送常见问题菜单。

当用户咨询额度，可以说明初始额度由系统综合评估，不建议承诺一定提额。

当用户咨询复借，可以说明复借会重新审核，不建议承诺一定通过。

当用户咨询利率，可以说明利率与信用表现相关，不建议承诺固定降息。

当用户发送投诉或高风险内容，应优先转人工处理，不要让 AI 独自给出承诺性答复。
